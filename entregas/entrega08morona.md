# Plano Individual de Trabalho

Cada estudante deve entregar uma compilação das tarefas realizadas até o momento no projeto, bem como um plano de trabalho para as próximas semanas (até a semana de 22 de Novembro).

O que cada tarefa deve conter:

- Título da tarefa
- Descrição da tarefa
- Data de início e término
- Área de conhecimento: front-end (front-end, back-end, banco de dados, etc.)
- Status da tarefa (iniciada, em andamento, concluída)
- Observações adicionais
